1. Download terlebih dahulu dataset of experiments
2. Buat drive untuk menyimpan dataset
3. Ubah alamat drive pada algoritma bagian "IMPORT LIBRARY" sesuai dengan alamat dataset yang anda buat


1. Kombinasi menggunakan Convolutional Neural Network (CNN) dengan fitur ekstraksi Short-Time Fourier Transform (STFT).
   - Jalankan CNN_with_STFT.ipynb
2. Kombinasi menggunakan Convolutional Neural Network (CNN) dengan ekstraksi Mel-Frequency Cepstral Coefficients (MFCC).
   - Jalankan CNN_with_MFCC.ipynb
3. Kombinasi menggunakan Long-Short Term Memory (LSTM) dengan fitur ekstraksi Short-Time Fourier Transform (STFT).
   - Jalankan LSTM_with_STFT.ipynb
4. Kombinasi menggunakan Long-Short Term Memory (LSTM) dengan ekstraksi Mel-Frequency Cepstral Coefficients (MFCC).
   - Jalankan LSTM_with_MFCC.ipynb
